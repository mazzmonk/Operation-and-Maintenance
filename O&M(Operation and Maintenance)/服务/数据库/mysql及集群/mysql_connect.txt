mysql-connector-c++-1.1.3安装

mysql-connector是官方提供的用于连接mysql的api函数库

1.下载使用源码包
mysql-connector-c++-1.1.3.tar.gz

设置环境变量
export MYSQL_INCLUDE_DIR=$HOME/apps/mysql/include/mysql
export MYSQL_LIB_DIR=$HOME/apps/mysql/lib
export MYSQL_DIR=$HOME/apps/mysql


$tar xxx.tar.gz
$cd mysql-connector-c++-1.1.3
$cmake .
$make

拷贝目录下的cppconn,driver目录到/usr/include/
拷贝driver/libmysqlcppconn* /usr/lib/下

2.在eclipse的c/c++ build->Settings->GCC C++ Linker->Libraries的右侧Libraries(-l)中添加
mysqlcppconn

保存

3.在编译时候头文件

#include <iostream>
using namespace std;

#include <driver/mysql_connection.h>
#include <driver/mysql_driver.h>
#include <cppconn/driver.h>
using namespace sql;
