Classes
class AggregationMethod：列出用于合并梯度的聚合方法的class.
class CriticalSection：临界区.
class DType：表示张量中元素的类型.
class DeviceSpec：表示 TensorFlow 设备的(可能是部分)规范.
class GradientTape：记录用于自动微分的操作.
class Graph：以数据流图表示的 TensorFlow 计算.
class IndexedSlices：给定索引处的一组张量切片的稀疏表示.
class IndexedSlicesSpec：tf.IndexedSlices 的类型规范.
class Module：基础神经网络模块 class.
class Operation：表示对张量执行计算的图节点.
class OptionalSpec：tf.experimental.Optional 的类型规范.
class RaggedTensor：表示不规则张量.
class RaggedTensorSpec：tf.RaggedTensor 的类型规范.
class RegisterGradient：用于为操作类型注册梯度函数的装饰器.
class SparseTensor：表示稀疏张量.
class SparseTensorSpec：tf.sparse.SparseTensor 的类型规范.
class Tensor：tf.Tensor 表示元素的多维数组.
class TensorArray：包装动态大小、按时间步长的 Tensor 数组的class.
class TensorArraySpec：tf.TensorArray 的类型规范.
class TensorShape：表示 Tensor 的形状.
class TensorSpec：描述 tf.Tensor 的类型.
class TypeSpec：指定 TensorFlow 值类型.
class UnconnectedGradients：控制当 y 不依赖于 x 时梯度计算的行为.
class Variable：请参阅变量指南.
class VariableAggregation：指示分布式变量的聚合方式.
class VariableSynchronization：指示分布式变量的同步时间.
class Constant_initializer：生成具有常量值的张量的初始化器.
class Name_scope：定义 Python 操作时使用的上下文管理器.
class ones_initializer：生成初始化为 1 的张量的初始化器.
class random_normal_initializer：生成服从正态分布的张量的初始化器.
class random_uniform_initializer：生成服从均匀分布的张量的初始化器.
class zeros_initializer：生成初始化为 0 的张量的初始化器.
