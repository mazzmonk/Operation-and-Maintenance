intel® threading building blocks 多线程库的编译安装

下载地址：
https://www.threadingbuildingblocks.org/

目前的版本是TBB4.3版本
这里用的是源码的版本

tbb43_20141204oss_src.tgz

1.编译
cd tbb43_20141204oss
make all
编译成功以后会在目录中生成build目录
同时会有include目录
cd include
ln -s /x/x/x/tbb /usr/include/
退到include同级目录
cd build
这里有2个这个目录
#linux_intel64_gcc_cc4.8_libc2.19_kernel3.13.0_debug
#linux_intel64_gcc_cc4.8_libc2.19_kernel3.13.0_release
#cd linux_intel64_gcc_cc4.8_libc2.19_kernel3.13.0_release


其中可以修改其中的内容
#export TBBROOT="/usr/include/tbb
#soure tbbvars.sh

最后编译自己写的代码的时候，使用-ltbb -lrt的选项
